用ChatGPT写了一个小项目，用来分析一个文件夹内所有的jpge文件信息，并进行统计。
主要用来统计目前所有拍摄照片中的焦段分布，为后续定焦镜头或新的变焦镜头的选择提供参考。
你也可以分享相关信息到社交媒体。
